# wsg
Wessaging Service  
"What's Good"  
"Wusgood"  
"WSG"  
  
This is designed to be a basic encrypted messaging service wih privacy in mind. My ultimate goal with this project is to create a fun way to commnicate with my friends, as well as to gain applicable experience with distributed systems, networking, and security. I am pair-programming with Claude Code, treating it as a mentor while I am its annoying intern, asking infinite questions.  
  
I eventually hope to have a basic GUI for this, but it is currently confined to the terminal to make me feel like a cool hacker.  
  
    
# Usage
run `$ make client`  
run `$ ./src/client <server ip>`  
enter your username then desired recipient as prompted  
to quit, type `/quit` in the message box  
to change recipients, type `/chat <recipient>` in the message box
